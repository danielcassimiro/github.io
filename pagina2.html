Muitas vezes é interessante definir algumas propriedade para cada elemento da página. o HTML possui diversos tipos de elementos, como listas, tabelas, imagens, forms, frames, etc.
Neste caso, iremos criar uma página padrão, utilizando apenas comandos do HTML.
Cada elemento no HTML, possui seus atributos como cor, fundo, estilo de texto, tamanho de caixa. Estas propriedades, podem ser tanto programadas dentro da propria linguagem, definindo para cada atributo um valor, na chamada deste, como programadas dentro de um arquivo CSS (Cascading Style Sheets).
O uso de css é interessante quando se deseja manter padrões no site, como por exemplo, usar as mesmas definições para todos os botões de um site, ou uma fonte para todos os textos específicos. Mas também, pode ser feitos estilos para determinados ID de cada elemento.

Estes são ótimos materiais para consulta.
